# 📱 UIKitPractice

A collection of hands-on iOS projects built using **Swift** and **UIKit** to practice fundamental and advanced iOS development concepts.

This repository demonstrates building applications completely with **Programmatic UI**, consuming REST APIs, following the **MVVM architecture**, and applying clean coding practices.

---

# 🚀 Projects Included

| Project                 | Description                                |
| ----------------------- | ------------------------------------------ |
| 🧮 CalculatorTest       | Calculator logic and unit testing practice |
| 🔢 MatrixController     | Matrix operations and controller practice  |
| 📱 MobileApp            | Smartphone product listing using REST API  |
| 🎬 MovieProject         | Movie browsing app using TMDB API          |
| 📋 UITableView Practice | Custom TableView and Cell implementation   |
| 🧭 Navigation Practice  | UINavigationController examples            |
